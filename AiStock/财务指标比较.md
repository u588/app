# 同行业企业比较中，判断企业优劣的核心财务指标

## ✅ 一、盈利能力（核心：赚钱能力）

> **衡量企业是否“赚得比别人多、赚得更稳”**

| Code | cnName | 原因 |
|------|--------|------|
| **col197** | 净资产收益率 | 衡量股东投入资本的回报率，是杜邦分析核心，反映企业综合盈利能力。ROE高通常代表管理效率高。 |
| **col199** | 销售净利率(%) | 每1元收入能带来多少净利润，反映成本控制和定价能力。 |
| **col202** | 销售毛利率(%) | 反映产品或服务的初始盈利空间，是行业竞争格局的直接体现。 |
| **col194** | 营业利润率 | 反映主营业务盈利能力，剔除投资等非经常性损益影响。 |
| **col96** | 归属于母公司所有者的净利润 | 最终归属股东的利润，是企业价值创造的终极体现。 |
| **col1** | 基本每股收益 | 直接反映每股股票的盈利贡献，是投资者最关注的指标之一。 |
| **col281** | 加权净资产收益率(每股指标) | 更精确地反映股东权益回报，尤其适用于有增发/回购的企业。 |

---

## ✅ 二、运营效率（核心：资产周转与资源利用）

> **衡量企业“用同样资产能否创造更多收入”**

| Code | cnName | 原因 |
|------|--------|------|
| **col175** | 总资产周转率 | 反映企业整体资产使用效率，越高说明资产利用率越好。 |
| **col172** | 应收帐款周转率 | 衡量回款速度，周转越快，资金占用越少，风险越低。 |
| **col173** | 存货周转率 | 衡量存货变现速度，对制造业、零售业尤为重要。 |
| **col177** | 应收帐款周转天数 | 与周转率互补，天数越短越好。 |
| **col178** | 存货周转天数 | 天数越短，库存积压风险越小。 |
| **col179** | 流动资产周转率 | 反映流动资产（现金、应收、存货）的运营效率。 |
| **col176** | 固定资产周转率 | 对重资产行业（如制造、能源）至关重要，衡量固定资产创收能力。 |

---

## ✅ 三、偿债能力（核心：安全边际与财务风险）

> **衡量企业“能否按时还钱、是否有破产风险”**

| Code | cnName | 原因 |
|------|--------|------|
| **col210** | 资产负债率(%) | 最核心的杠杆指标，过高意味着财务风险大，过低可能保守。 |
| **col159** | 流动比率(非金融类指标) | 衡量短期偿债能力，一般 >2 较安全。 |
| **col160** | 速动比率(非金融类指标) | 剔除存货后的流动性，更严格衡量短期偿债能力。 |
| **col161** | 现金比率(%) | 仅用现金及等价物覆盖流动负债，反映极端情况下的偿债能力。 |
| **col162** | 利息保障倍数(非金融类指标) | 衡量企业支付利息的能力，倍数越高越安全。 |
| **col166** | 有形资产净值债务率(%) | 反映有形净资产对债务的覆盖程度，适用于重资产企业。 |
| **col167** | 权益乘数(%) | 反映财务杠杆，乘数越高，风险越大。 |

---

## ✅ 四、现金流质量（核心：真金白银 vs 纸面利润）

> **衡量企业“赚的钱是不是真钱、能不能持续”**

| Code | cnName | 原因 |
|------|--------|------|
| **col107** | 经营活动产生的现金流量净额 | “造血能力”，是企业生存的根本。利润可操纵，现金流难造假。 |
| **col228** | 经营活动产生的现金流量净额与净利润比率 | 衡量利润的“含金量”，>1说明利润质量高。 |
| **col220** | 营业收入现金含量(%) | 收入有多少是真金白银收回的，反映销售回款质量。 |
| **col222** | 销售商品提供劳务收到的现金/营业收入 | 同上，直接反映收入真实性。 |
| **col223** | 经营活动产生的现金流量净额/营业收入 | 衡量每1元收入带来的经营现金流。 |
| **col226** | 经营净现金比率（短期债务） | 经营现金流对短期债务的覆盖能力。 |
| **col227** | 经营净现金比率（全部债务） | 经营现金流对总债务的覆盖能力。 |

## ✅ 五、成长性（核心：未来潜力）

> **衡量企业“增长是否可持续、能否扩大规模”**

| Code | cnName | 原因 |
|------|--------|------|
| **col183** | 营业收入增长率(%) | 核心成长指标，反映市场扩张能力。 |
| **col184** | 净利润增长率(%) | 反映盈利增长速度，但需结合现金流验证。 |
| **col185** | 净资产增长率(%) | 反映股东权益积累速度，是内生增长能力的体现。 |
| **col186** | 固定资产增长率(%) | 对重资产行业，反映产能扩张意愿。 |
| **col187** | 总资产增长率(%) | 反映企业整体规模扩张速度。 |
| **col190** | 扣非每股收益同比(%) | 剔除非经常性损益后的真实增长，更客观。 |
| **col191** | 扣非净利润同比(%) | 同上，反映主营业务增长质量。 |

## ✅ 六、股东回报与估值相关（核心：为股东创造价值）

> **衡量企业“是否善待股东、股价是否有支撑”**

| Code | cnName | 原因 |
|------|--------|------|
| **col1** | 基本每股收益 | 投资者最直观的收益指标。 |
| **col2** | 扣除非经常性损益每股收益 | 更真实反映主营业务盈利能力。 |
| **col3** | 每股未分配利润 | 反映企业累计留存利润，是未来分红或转增股本的基础。 |
| **col4** | 每股净资产 | 估值基础，市净率 = 股价 / 每股净资产。 |
| **col5** | 每股资本公积金 | 反映企业资本积累，可用于转增股本。 |
| **col219** | 每股经营性现金流 | 每股对应的“真金白银”，是分红能力的基础。 |
| **col225** | 每股现金流量净额 | 包含投资/筹资活动，反映整体现金创造能力。 |
| **col238** | 总股本 | 用于计算EPS、ROE等每股指标。 |
| **col242** | 股东人数(户) | 人数减少可能预示筹码集中，增加机构持仓比例。 |
| **col243** | 第一大股东的持股数量 | 反映股权集中度，过高可能有治理风险，过低可能缺乏稳定性。 |
| **col246** | 机构总量（家） | 机构参与度高，通常代表基本面被认可。 |
| **col247** | 机构持股总量(股) | 机构持仓比例高，通常意味着公司受专业投资者青睐。 |

## 🔍 特别提示

- **避免只看单一指标**：例如，高ROE可能来自高杠杆（风险），高营收增长可能伴随负现金流（不可持续）。
- **结合行业特性**：
  - 制造业：重视存货周转率、毛利率、固定资产周转率。
  - 服务业：重视人均创收、销售费用率、现金流质量。
  - 金融业：部分指标不适用（如流动比率、速动比率），需用资本充足率、不良贷款率等替代。
- **动态比较**：不仅要看当前值，还要看趋势（如连续3年ROE是否提升、现金流是否改善）。
- **异常值排查**：如“营业外收入”、“非经常性损益”占比过高，需警惕利润“注水”。

---

✅ **总结**：

在同行业比较中，最能体现企业优劣的财务项目是：

> **“盈利能力+现金流质量+成长性”三位一体** ——  
> 一个好企业 = **赚得多（ROE高）、赚得真（现金流强）、长得稳（营收利润双增长）**
