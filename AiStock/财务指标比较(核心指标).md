### ✅ 精简原则（特别强调可比性）

1. **优先使用比率、百分比、每股类指标**（不受股本/规模影响）；
2. **剔除所有绝对金额指标**（如“营业收入”“净利润”“存货”等），除非有标准化形式（如“毛利率”）；
3. **剔除仅适用于特定行业**（如金融类）的指标；
4. **避免重复表达同一经济含义**的多个指标；
5. **保留能反映质量、效率、风险本质的最小必要集合**。

在公司横向比较中，**使用最新、最完整、最具可比性的财务数据**至关重要。对于利润表和现金流量表相关的指标，**单季度（Single Quarter）或上年年报（Last Annual）数据往往存在季节性偏差或滞后性**，因此应优先采用 **TTM（Trailing Twelve Months，滚动12个月）** 数据进行计算。

### ✅ 一、需要 TTM 处理的指标（共 9 项）

以下指标**基于利润表或现金流量表**，其分子或分母涉及 **期间性数据（如收入、利润、现金流等）**，必须使用 **TTM 累计值** 才能保证跨公司、跨时点的可比性：

| 指标（字段） | 名称 | 是否需 TTM | 原因说明 |
|-------------|------|-----------|--------|
| **col281** | 加权净资产收益率（ROE） | ✅ 是 | ROE = 净利润 / 净资产。若净利润用单季或去年全年，会导致：① 季节性失真（如零售Q4高）；② 与当前市值/股价不匹配。**应使用 TTM 净利润 ÷ 最新季度净资产**（近似TTM ROE）。 |
| **col202** | 销售毛利率（%） | ✅ 是 | 毛利率 = （营收 - 营业成本）/ 营收。必须用 **TTM 营收与TTM营业成本** 计算，避免季度波动（如芯片行业Q2淡季拉低毛利）。 |
| **col199** | 销售净利率（%） | ✅ 是 | 净利率 = 净利润 / 营收，同样需 **TTM 净利润 / TTM 营收**。 |
| **col2** | 扣除非经常性损益每股收益 | ✅ 是 | EPS 应基于 **TTM 扣非净利润 / 最新总股本**，否则无法反映当前盈利水平。 |
| **col172** | 应收账款周转率 | ✅ 是 | = TTM 营业收入 / 期末应收账款。**分子必须TTM**，分母用最新季度末值（时点数）。 |
| **col173** | 存货周转率 | ✅ 是 | = TTM 营业成本 / 期末存货。同上，分子需TTM。 |
| **col175** | 总资产周转率 | ✅ 是 | = TTM 营业收入 / 期末总资产。分子TTM，分母时点值。 |
| **col228** | 经营活动现金流净额 / 净利润 | ✅ 是 | 分子（经营现金流）和分母（净利润）**都必须是TTM值**，否则比率失真（例如：单季现金流为负但全年为正）。 |
| **col222** | 销售商品收到现金 / 营业收入 | ✅ 是 | 两者均为期间流量，必须用 **TTM 经营现金流入（col101） / TTM 营业收入**。 |

> 🔸 注：虽然 col222 的分子是“销售商品收到的现金”（对应 col98），但在TTM计算中，应使用 **最近4个季度 col98 之和**。

### ❌ 二、**不需要 TTM 处理**的指标（共 7 项）

这些指标要么是**资产负债表时点数据的比率**，要么是**同比增速**，本身已具备可比性，无需TTM：

| 指标（字段） | 名称 | 是否需 TTM | 原因说明 |
|-------------|------|-----------|--------|
| **col160** | 速动比率 | ❌ 否 | = （流动资产 - 存货）/ 流动负债，全部来自**最新季度末资产负债表**，是时点快照，直接使用即可。 |
| **col210** | 资产负债率（%） | ❌ 否 | = 总负债 / 总资产，同样是**最新季度末时点数据**，无需累计。 |
| **col183** | 营业收入增长率（%） | ❌ 否 | 本身就是 **（本期TTM营收 - 上年同期TTM营收）/ 上年同期TTM营收**，标准同比增速，已隐含TTM逻辑。 |
| **col184** | 净利润增长率（%） | ❌ 否 | 同上，标准同比指标，通常由数据源直接提供（基于TTM或年度滚动）。 |
| **col191** | 扣非净利润同比（%） | ❌ 否 | 同上，属于增长率类，已标准化。 |
| **col238** | 总股本（亿股） | ❌ 否 | 是**当前最新股本**（时点数），用于市值计算，无需TTM。 |
| **自由流通股比例**<br>（= col266 / col238） | 自由流通占比 | ❌ 否 | 两者均为最新时点数，直接相除即可。 |

> 💡 补充说明：  
> - **增长率类指标**（如 col183/184/191）虽然底层依赖TTM数据，但**最终输出已是标准化同比值**，使用者无需再处理TTM；  
> - **资产负债率、速动比率**等纯资产负债表比率，天然基于同一时点，直接可比。

### ✅ 三、实操建议：如何获取 TTM 数据？

1. **若你有季度财报数据库**：
   - 对每个公司，取 **最近4个季度的利润表/现金流量表项目**，求和得到 TTM 值；
   - 资产负债表项目（如应收账款、总资产）使用**最新季度末值**。

2. **若使用第三方数据接口（如Tushare、Wind、聚宽）**：
   - 优先调用字段如 `revenue_ttm`、`netprofit_ttm`、`operating_cashflow_ttm`；
   - ROE_TTM、毛利率_TTM 等常已预计算，可直接使用。

3. **注意财报发布时间差**：
   - 比较时应统一截至**相同报告期**（如都用 2025Q3 报告期的TTM数据），避免“用A公司2025Q3 vs B公司2025Q2”。
