## 三、四阶筛选法（2025升级版）


### 1、 统计显著性过滤

* 严控 `fdr_level=0.001`（消费股噪声低于周期股）  
* 剔除IC值<0.05的特征（如`variance_larger_than_std`）

### 2、消费逻辑验证

* 正向关系：品牌溢价特征 vs 毛利率 (`spearmanr > 0.3`)  
* 负向关系：价格弹性特征 vs 销量增速 (`spearmanr < -0.2`)  
* 陷阱案例：`kurtosis`（峰度）在消费股中多为噪音

### 3、政策稳定性测试（关键！）

```python
   # 2025年消费刺激政策断点检验 
   policy_dates = ['2025-03-01','2025-06-18']
   stability_score = []
   for date in policy_dates:
       pre_corr = features.loc[:date].corr(target)
       post_corr = features.loc[date:].corr(target)
       stability_score.append(1 - (post_corr - pre_corr).abs().mean())
   # 保留稳定性>0.8的特征 
   robust_features = features.columns[stability_score > 0.8]
   ```
