---------------- 来获取一个临时的 token, 默认 1 小时 ---------
kubectl create token kubeapps-operator -n kubeapps  --duration  

--------------------- 手动长期有效的 API 令牌 ----------- 
kubectl apply -f - <<EOF
apiVersion: v1
kind: Secret
metadata:
  namespace: kubeapps
  name: kubeapps-operator-secret
  annotations:
    kubernetes.io/service-account.name: kubeapps-operator
type: kubernetes.io/service-account-token
EOF
